# Projectfy MVP

Este bootstrap segue o escopo inicial do `projectfy_roadmap.md`.

## Entregue

- Frontend React + TypeScript + Vite + TailwindCSS
- React Router com rotas de auth, admin e portal do cliente
- Zustand para alternancia de tema
- React Query configurado para futura camada de API
- React Hook Form + Zod no login
- Backend NestJS com Swagger e módulos iniciais
- Prisma schema com enums e modelos centrais
- Docker Compose com frontend, backend, postgres, redis e nginx

## UI/UX

As referencias visuais foram traduzidas para um produto SaaS denso, com sidebar, topbar, tabelas, cards de KPI, paineis translúcidos e portal do cliente.

Paleta aplicada:

- Tema claro: branco, azul marinho, azul claro e verde
- Tema escuro: azul marinho, azul claro, cinza e laranja
